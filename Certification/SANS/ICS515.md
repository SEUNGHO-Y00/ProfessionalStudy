### ICS Protocols

* Protocol Payloads
* Modbus TCP Breakout
  - Modbus Port Details
    - IANA Assigned: The Internet Assigned Numbers Authority (IANA) explicitly reserved port 502 for Modbus protocol traffic.
    - Cleartext Traffic: Traffic sent over port 502 is completely unencrypted. Anyone sniffing the network can read the register values and register modifications in plain text.
    - No Authentication: Port 502 accepts commands from any IP address that can reach it. An attacker can send raw hex commands directly to this port to turn off equipment or alter industrial setpoints.

* How to Install Zeek
  - Because Zeek is primarily built for Unix-like systems, installation methods vary by platform. Below are the most direct ways to install it.

1. Linux (Ubuntu / Debian)
  - For modern Linux distributions, the easiest method is using the binary packages provided by the OpenSuSE Build Service hosted by the Zeek project.
  - Open your terminal and run:
```bash
# Update your repository list
sudo apt update

# Install dependencies needed for repositories
sudo apt install -y curl gnupg2 software-properties-common

# Add the official Zeek repository key (Example for Ubuntu 22.04/24.04)
curl -fsSL https://opensuse.org | sudo gpg --dearmor -o /etc/apt/trusted.gpg.d/security_zeek.gpg

# Add the repository to your sources list
echo 'deb http://opensuse.org /' | sudo tee /etc/apt/sources.list.p.d/security:zeek.list

# Install the stable release of Zeek
sudo apt update
sudo apt install -y zeek-lts
```

Note: Make sure to add /opt/zeek/bin to your system's PATH variable after installation to run zeek from any directory.

2. macOS
  - The fastest and most reliable way to install Zeek on a Mac is via the Homebrew package manager.
  - Open your terminal and run:
```bash
# Install Zeek using Homebrew
brew install zeek
```
